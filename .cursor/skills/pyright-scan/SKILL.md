# Последовательность действий

1. Найти и запустить скрипт `.scripts/pyright-scan.py`.
2. Использовать `python .scripts/pyright-scan.py --format json`.
3. Дождаться завершения и проверить, что отчёт сохранён в `docs/reports`.
4. Проанализировать сохранённый JSON-отчёт.
5. Исправить все найденные в репорте ошибки и предупреждения в коде проекта.
6. Повторно запустить `pyright-scan.py` и убедиться, что отчёт не содержит новых замечаний.

## Примечания

- Отчёт должен быть размещён в `docs/reports`.
- Работа ведётся только по текущему проекту в корне репозитория.
- Если сканирование завершилось с ошибкой выполнения, сначала устранить причину запуска, затем повторить проверку.
- Кол-во повторов не более 3 раз. Если после 3-х повторов отчёт всё ещё содержит ошибки, необходимо остановиться и запросить дальнейшие инструкции у пользователя.
