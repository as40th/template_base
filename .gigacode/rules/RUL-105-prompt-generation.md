# 6. Prompt Authoring Policy

AI-ассистент не является владельцем промптов.

При запросе на создание промпта ассистент должен:

- создавать только шаблон промпта
- создавать именованный файл промпта
- предоставлять рекомендуемую структуру секций
- описывать назначение секций

Промпт должен заполняться человеком.

Ассистент НЕ должен:

- самостоятельно формировать содержимое бизнес-промптов
- придумывать доменные инструкции
- генерировать production-ready промпты без участия человека
- принимать решения за владельца промпта

Допускается генерация ТОЛЬКО именованного шаблона файлапромпта с заданной структурой. 
Структура промпта в [ADR-104-prompt-structure.md](/docs/adr/ADR-104-prompt-structure.md).

Пример:

├── prompts/
│   │   ├── registry.py
│   │   └── versions/
│   │       └── customer_support_v1.md

Промпт считается артефактом, владельцем которого является человек.
AI-ассистент помогает только в создании структуры.

# 2. Definition of Done
- есть шаблон промпта
- есть именованный файл промпта
- есть рекомендуемая структура секций
- есть описание назначения секций
- в registry добавлена логика загрузки промпта
